This project uses Electron + Svelte 5 + fabric.js.
It uses Svelte 5's new runes syntax, like `$state`etc.
If you have any question about Svelte, you can find the full documentation in the file svelte-docs-for-llms.md

You will notice the dependencies contain a package
called runed. Runed contains some useful utilities like `StateHistory`.
However, it is currently an unused dependency.
